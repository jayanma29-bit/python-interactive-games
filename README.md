# python-interactive-games
This was my very first python project made which focuses on core concepts such as functions, loops, variables, input handling, conditionals, and much more. 

---
##  Games Included

| # | Game | Description |
|---|------|-------------|
| 1 | **Caesar Cipher** | Encrypt or decrypt a message using a letter and shift key |
| 2 | **Number Guessing** | Guess a random number between 1–100 with hints |
| 3 | **Even or Odd** | Check whether any number is even or odd |
| 4 | **Coordinate Difference** | Calculate the distance between two points on a coordinate plane |
| 5 | **Game of Pig** | Dice game vs. the computer the first to 100 points wins  race to 100 points |
| 6 | **Tic Tac Toe** | Traditional tic tac toe game | 

##  What I Learned

- Defining and calling functions
- Handling user input and validating it with try/except
- Using while loops for game logic
- Working with Python's random and math modules
- Building a simple menu-driven program flow
